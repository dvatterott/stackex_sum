Sifting the Overflow

Sifting the Overflow is designed to identify the most helpful portions of stackoverflow answers to questions about the programming language python. The code described in this tutorial is available on my github page.

If you want to use the chrome extension, follow this link to the google chrome store where you can install it. To use Sifting the Overflow, press the button when on a stackoverflow page with a question, and the Sifting the Overflow will highlight the most helpful answers.

If you don't want to install the browser extension of a random person on the web (me), you can also copy and paste the url of a stackoverflow question into the dialog box below, which will produce the same result as using the browser extension.
