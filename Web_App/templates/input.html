Highlight helpful parts of Stack Overflow answers

Sifting the Overflow is designed to help novice programmers identify the most helpful portions of Stack Overflow answers about the programming language python. The code described in this post is available in my github repository.

If you want to use the Chrome extension, follow this link to the Google Chrome store where you can install it. To use Sifting the Overflow, press the button when on a Stack Overflow page with a question, and Sifting the Overflow will highlight the most helpful sentences.

If you don't want to install the browser extension, you can also copy and paste the URL of a Stack Overflow question into the dialog box below, which will produce the same result as using the browser extension.
